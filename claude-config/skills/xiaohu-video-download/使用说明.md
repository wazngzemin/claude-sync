# 视频下载工具 — 使用说明（给人类用户）

怎么用（告诉 Claude）
- “这个链接下载视频/音频” → 自动按默认路径保存。
- “下载这个播放列表，前5个/全部” → 按要求批量保存。
- “给本地视频加字幕：<文件路径>” → 自动转写/翻译/烧录。
- “下载这个抖音视频 <链接>” → 需保证抖音已登录（见下）。
- 首次未配置输出目录时，脚本会询问想要把输出目录设置在哪里，请输入绝对路径，后续会自动处理

能做什么
- 单个视频：下载视频 + 外挂字幕（无字幕则只下载视频，外语字幕可翻译成中文）。
- 播放列表：批量下载，可选范围/分段，文件名自带序号。
- 仅音频：提取音频并写入封面/元数据。
- 本地视频加字幕：无字幕先转写/翻译，再烧录；已有字幕可直接烧录。
- 抖音下载：支持 modal_id/短链，登录状态下抓取。

YouTube 近况（为什么有时会失败）
- 近一年 YouTube 的反爬/风控更严格，`yt-dlp` 在无 cookies 的情况下更容易遇到 `HTTP Error 403` / SABR / PO Token 等限制。
- 解决思路：失败后改用浏览器（已登录/已同意隐私弹窗）的 cookies 重试。

输出位置
- 默认：读取本技能 `config.json` 的 `output_dir` 作为 `<输出根>`，最终写入 `<输出根>/data/`（最终文件）与 `<输出根>/tmp/`（临时）。
- 自定义：用配置面板或直接编辑 `config.json` 的 `output_dir`，或在指令/命令中带 `--outdir <绝对路径>`（命令行优先）。务必用绝对路径，避免写到当前目录的 `data/`。

抖音提示
- 首次或登录过期：运行 `python3 scripts/douyin_login.py`，浏览器登录后回车保存；状态在 `data/browser_state/douyin_profile/`。
- 下载失败请先看 `抖音初始化.md`，检查登录、网络/代理，再重试。

输出文件在哪里
- 视频/音频/字幕：在 `<输出根>/data/`。
- 临时文件：在 `<输出根>/tmp/`（可自行清理）。

实用命令（可直接跑）
- YouTube 下载（失败自动用浏览器 cookies 重试，推荐）：
  `python3 scripts/youtube_download.py "https://www.youtube.com/watch?v=XXXX"`
