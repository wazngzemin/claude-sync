[entities] 种子数据（Seed Data）
种子数据（Seed Data）
定义
种子数据是为车载 AI 助手（planner）构造的高质量结构化数据集，用于评测和数据泛化。每条数据包含用户 query、Context 字段和期望输出。
用途
评测：作为评测集验证 Planner 的意图理解和工具调用能力
数据泛化：作为基础数据，通过泛化生成更多训练/评测样本
数据结构
每条种子数据包含以下字段：


字段

说明

用户 Query

用户说的话（可能多轮）

视觉信息

时间、地点、天气

舱内情景描述

静态（外貌/物品）+ 动态（行为/情绪）

舱外情景描述

静态（环境）+ 动态（人员活动/信号变化）

端状态

车窗、空调、座椅、氛围灯等车辆状态

用户记忆

饮食偏好、社会关系、地点记忆等

Goal List

当前目标、细节和进展（JSON 格式）

人音区映射

各座位乘客姓名

说话人

当前 query 的说话人

车辆知识

车辆相关知识的格式化描述（格式待定）
覆盖场景
种子数据按 Planner 能力划分为：像人一样对话、明确/模糊车控、人脸注册和记忆操作、POI 检索/路径规划/导航、内容媒体搜推、信息查询问答、主动服务（Advisor）、跟进任务、高阶体验。
