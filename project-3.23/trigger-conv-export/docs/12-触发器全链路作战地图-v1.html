Part 1完整主链路 —— 从用户 query 到最后输出

整条链路其实是两条独立的流在触发器这里交汇:一条是感知上行流(车和摄像头持续把状态送上云),一条是query下行流(用户说话被逐层处理)。触发器⑮站在两条流的交点——它读上行流的信号,判断query下行流注册的条件成不成立。

绿底=你的节点蓝=别人的节点红=卡点(见Part 3)
