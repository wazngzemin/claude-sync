# Article Forks 索引

按时间倒序排列。每次 fork 新文章时自动追加一行。

## 检索方式

- **按日期**：文件夹名以 `YYYY-MM-DD-` 开头，直接按名称排序即可
- **按分类**：见下表 `分类` 列
- **按关键词**：在本文件中搜索

## 分类体系

| 标记 | 含义 | 说明 |
|------|------|------|
| `方法论` | 有明确可执行步骤的文章 | 会生成教程 + 工具 |
| `框架` | 提出分析框架或思维模型 | 会生成教程 + 分析模板 |
| `访谈` | 访谈、播客、对话记录 | 会提取多个概念页 |
| `案例` | 某人/某公司怎么做某事 | 会生成可复用模式教程 |
| `观点` | 纯观点/叙事 | 只生成 source 摘要 |
