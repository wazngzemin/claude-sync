# 小区图书馆管理系统 PRD

> 版本 V1.1 | 创建日期 2024-01-20 | 状态：已确认

## 概要
为小区图书馆打造的轻量级图书管理系统，纯前端实现（HTML5 + CSS3 + JS），数据存储在 LocalStorage。

## 核心功能
1. **图书管理** — 添加/编辑/删除图书（书名、作者、楼层、类型、价格）
2. **借书人管理** — 注册/编辑借书人（身份证、姓名、手机号、押金）
3. **借阅管理** — 借书/还书，自动计算逾期天数和扣费
4. **首页仪表盘** — 待还书数量、逾期高亮、借阅历史

## 技术架构
- 纯前端：原生 HTML5 + CSS3 + JavaScript
- 存储：LocalStorage（约 5MB 限制）
- 无服务器依赖，单页应用

## 关键业务规则
- 逾期费用 = 逾期天数 × 10 元/天，从押金中扣除
- 押金不足（<0）时禁止借书
- 图书状态流转：available → borrowed → available
