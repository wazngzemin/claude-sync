泽民（王泽民）反复问的深层问题不是"某场会怎么复盘"，而是"为什么我这么专业勤奋、产出这么多，却在评审里一场场被架空"。2026-07 一次深谈中给出的根本诊断：

**核心：他是"执行型PM"——把事做对(execution)很强，但"定义对的事+施加影响"(definition+influence)几乎空白。** 勤奋本身成了高级的逃避：打磨文档安全可控(他说了算)，评审桌正面博弈危险失控(可能当众错)，于是他的精力不知不觉全流向前者。

**证据(不可反驳，全是他的事实):** 触发器PRD做到v12、讲解稿v9(产出过剩) vs 7.2/7.4/7.8连续丢定位/丢归属/被推翻(博弈极弱)。

**三个稳定短板:** A定义权(只回答别人定义的问题，从不定义议题) / B归属谈判(接责不要对价、撤退不留条款) / C押注判断(敢在PRD里下判断，一被当面顶就退"那还是按原来")。

**为什么(三层):** ①路径依赖闭环(执行有正反馈、博弈有负反馈→越躲越弱，锁死中级) ②安全感来源错了(建立在"产出够好"=优等生心态，而非"判断够对且敢担责"=主人心态) ③赌注太高反而不敢下注(触发器=转具身智能门票，太怕出错→选"不争"的打法→慢慢失去它)。

**How to apply(对我的约束):**
- 别再无脑给他做又一版产出。他要HTML/复盘时，先判断是不是又在"用产出逃避正面博弈"，该推他上桌就推。
- 所有诊断/建议从三个短板切入，落到"下一场会的具体动作"，不给抽象道理、不灌鸡汤(违禁词见 [[feedback-review-coaching-framework]])。
- 元缺口=他缺"敢对模糊+有人反对的事拍板并负责"本身；框架都是脚手架，最后那一下(张嘴/押判断/扛反对)只能靠次数脱敏。
- 升高级PM=三次重心转移(做对事→定义对的事 / 完成任务→扛模糊与责任 / 讲清方案→让人按我方向走)；第1项有底子不敢押注，2/3空白。这一步与转具身智能是同一步。
