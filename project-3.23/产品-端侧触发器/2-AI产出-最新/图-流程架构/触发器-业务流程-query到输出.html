🔎 端云怎么分 + 流向怎么走（看橙色粗线＝跨端云）：
端侧做的：① 用户query / ② ASR / 车控快路径(fc+白名单+原子车控) / TTS播报+卡片 / 机器人 / 车控执行 / 端信号采集。
云端做的：③ 拒识仲裁 / 意图解析+注册场景 / Director 推理(+Context) / 操作工具(部分) / 信号库+触发器+advisor。
跨端云（橙线）：ASR→拒识仲裁(端→云) · 拒识→车控fc(云→端) · Director→TTS/机器人/车控(云→端) · 端信号→云信号库(端→云)。
🎯 例子（"我好热啊"）：端[说→ASR] →端→云→ 云[拒识仲裁=非车控→Director(注入Context:车内28°)→推理→talk+action] →云→端→ 端[TTS:"这就帮你降温"+卡片 / 车控:开空调风量3档26°]。 一句话跨端云走一圈，说做一致。
