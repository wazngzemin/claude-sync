## 摘要

李广密判断 AGI 将在 2 年内实现，给出三幕路线图（Chatbot → Coding Agent → 自动化 AI 研究员）。核心洞察：Coding 不是编程能力，而是一个"环境"——代码是对解决方案的描述，模型通过生成和执行代码来与外部世界交互。同时对 OpenAI（战略分散+组织动荡）和 Google（营销成功但 Coding 严重落后）做了尖锐判断。

## 关键要点

- **AGI 路线图三幕**：Chatbot（第一幕，已完成）→ Coding Agent（第二幕，正在发生）→ 自动化 AI 研究员（第三幕，AGI）
- **Coding 是环境不是能力**：自然语言描述世界，代码描述解决方案。模型通过代码实现对外部信息的采集/处理/反馈
- **人工写代码已 <1%**：硅谷研究员和顶级公司程序员现在是审核AI代码或管理Agent集群
- **OpenAI 的问题**：ChatGPT 成功导致专注 ToC 忽视 Coding；资源分散到 Sora 等支线；核心团队持续流失（Anthropic/Ilya/Mira）
- **OpenAI 的真正使命**：不应把构建互联网平台当最高目标，push 智能才是；过早追求商业可能错过 AGI 科学的"文艺复兴"
- **Google 被高估**：Gemini 3.0 营销成功但实际体验不及预期；Coding 严重落后是最大战略失误；是"最领先的追随者"但跟随者拿不到最大蛋糕

## 逐段拆解

### 第一段：AGI 路线图（"三幕论"）

**功能**：核心判断 + 时间框架

**深度分析**：三幕论的精妙在于每一幕的关系是"前一幕是后一幕的基础设施"——Chatbot 证明了语言理解能力，Coding Agent 把理解力变成执行力，自动化 AI 研究员则是执行力指向自身（AI 改进 AI）。"2年内"的判断隐含一个假设：从 Coding Agent 到自动化研究员的跳跃不需要新范式。

**隐含假设**：Coding Agent 的成熟度足以支撑 AI 自我改进；不会出现"最后一英里"问题。

### 第二段：Coding 是环境（核心洞察）

**功能**：重新定义概念

**深度分析**：这是全文最有价值的认知升级。把 Coding 从"技能"重新定义为"环境"（environment），对应强化学习的框架——Agent 在环境中行动、获得反馈、学习。自然语言描述世界（被动），代码描述解决方案（主动）。这意味着：会写代码的 AI ≠ 只是程序员，而是 = 拥有了改变世界的操作接口。

**隐含假设**：现实中大多数有价值的任务都可以被形式化为代码可执行的形式。

### 第三段：OpenAI 的问题（"战略分散 + 组织动荡"）

**功能**：竞争格局分析

**深度分析**：把 OpenAI 的问题归结为两层——（1）战略层：ChatGPT 的 ToC 成功让他们"赢了一场战役但可能输掉战争"，因为 Coding 才是通往 AGI 的关键路径；（2）组织层：自下而上创新文化孕育了 ChatGPT 但也导致分散。"几千人改变几十亿人"的杠杆判断非常清醒——但他质疑 OpenAI 是否把这个杠杆用在了最高价值的方向。

**隐含假设**：Anthropic（由 OpenAI 出走者创建）在 Coding 方向上走得更对。

### 第四段：Google 判断（"最领先的追随者"）

**功能**：竞争格局补完

**深度分析**："最领先的追随者"是极其精准的定位——有资源、无短板、但无原创方向。Benchmark 过度优化的指控对产品经理有直接启示：不要被 benchmark 骗了，看用户真实体验。"掉队可能性很低但拿不到最大蛋糕"暗示了一个规律：在范式转换期，快速跟随者的天花板低于方向定义者。

**隐含假设**：Coding 领先 = AGI 领先，这个等式成立。

## 论证链

"AGI 2年内实现"（时间判断）→ "路线：Chatbot→Coding Agent→AI研究员"（路径）→ "Coding 是环境不是技能"（重新定义）→ "代码=解决方案的描述语言"（原理）→ "OpenAI 战略分散错失 Coding"（竞争分析）→ "Google 是跟随者拿不到最大蛋糕"（格局）→ "push 智能才是最高目标"（价值判断）

## 隐含假设

1. Coding Agent → AI 研究员的跳跃不需要新范式突破
2. 代码能够形式化描述大多数有价值的现实任务
3. Coding 领先 = AGI 竞赛领先
4. 组织问题（人才流失）是 OpenAI 的根本制约而非模型能力
5. Anthropic 隐含地被认为走在更正确的路上（但未明说）

## 观点边界

1. 2 年时间线过于乐观——Hassabis 认为还缺 1-2 个突破
2. "代码能描述一切解决方案"忽略了物理世界的不可形式化部分
3. 对 Google 的判断可能低估了其在科学应用（DeepMind）方面的独特优势
