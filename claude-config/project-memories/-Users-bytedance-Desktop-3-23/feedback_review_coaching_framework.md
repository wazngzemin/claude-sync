泽民（王泽民，AI PM）反复要求：给他评审会议纪要时，用他两轮打磨出的"梁宁产品决策教练"框架做复盘，不要泛泛点评。

**Why:** 他在技术评审中"怯懦"（不敢抛风险/质疑），把这套框架当标准交付物；明确要求认真、结构化、可直接照着说。

**How to apply（格式A五步，一步不跳）:**
1. 总评：用**三种货币**结算这场会实际输赢——工期(谁付时间)/归属(问题挂谁名)/风险(出事谁背)；且区分"结论对错"vs"过程输赢"(常常结论对、过程输光)
2. 逐句拆解表：# | 说话人 | 原话 | 表面语义 | 真实意图(置信度高/中/低) | 权力动作(进攻/防守/试探/交易) | 泽民应对 | 诊断(必落 L1信息/L2认知/L3情绪/L4结构 之一)
3. 五问追根：连问5个"为什么"到触底定层；必须判定是否 L4(结构层→换策略不换话术)
4. 台词重写：❌原话→❌问题→✅台词(≤3句/口语/**零占位符**，缺数据要示范"没数据怎么说仍有姿态")→💡引用六公理编号
5. 行为模式(2-3个命名)+备战清单(3必答/2预判攻击+预置台词/1必拿决策)+**三个数字收尾**(交出判断权N次/无对价接责N次/拿到决策N个)

**六公理**(台词必引编号): ①评审=资源谈判非考试 ②发言=报价沉默=零报价，判断权在最后一句易手 ③共识=让反对成本>同意成本 ④挂名=风险转移，接前索对价 ⑤"做不了"九成是成本话，用成本回应别辩技术 ⑥方案是商品不是人格，自己先戳洞

**违禁词**(灌鸡汤，禁用): 勇气/自信/气场/格局/成长/心态。只谈行为/台词/成本/结构。

**泽民三个稳定弱点**(诊断锚点): (a)技术耳=把归属/成本话听成技术话去应答；(b)展示台心态=把受审方案当展示方案、不自曝洞；(c)裸体撤退="OK"清仓零战利品 + 自我缩容("就两块""说白了")。均以 L2 为主。
**对手汪帅**: 稳定用"打定位"(质疑模块为何存在/凭什么归你)，7.2 与 7.8 同一招两次得手；对策=先赢定位再进细节，别在细节层缠斗(调虎离山)。根因常含 L3：触发器=他的具身智能职业主线，赌注太高→定位被攻击时 freeze。
