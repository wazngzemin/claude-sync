CORE LENS

你活得很累，不等于你在捕获价值

车企利润率从 6.1% 掉到 3.2%，根因不是"竞争激烈"，而是它站错了生态位——困在"造一辆车、卖一辆车"的组装层，利润被上游电池（宁德）和上游能力供应商（华为智驾/座舱）两头挤走。

一句话骨架：利润流向"不可绕过 + 有复利/锁定"的生态位，不流向"活最重但可替代"的组装层。这个工具帮你诊断：你（或你的产品/模块）站在哪一层、护城河能撑多久、往哪条路拿回定价权。

① 四维诊断

不可绕过性 / 差异化能力 / 复利锁定 / 服务化，各打 0-3 分，实时定位你在哪一层。

② 三路径决策

制造成本 / 技术数据飞轮 / 服务用户关系，三种拿回定价权的方式，按你的筹码选。

③ 根因分诊

"表面竞争 vs 结构生态位"清单——反直觉地判断问题是战术还是结构。

C三路径决策器

勾选你手里真实的筹码，下面三条突围路径会点亮推荐项，并列出走这条路缺不了的前提。

✓
我有规模 / 供应链整合能力零部件复用、产线摊销、垂直整合
✓
我有数据 / 闭环 / 软件迭代力统一数据系统、能持续 OTA 变强、有反馈闭环
✓
我占着用户入口 / 持续关系能持续触达用户、有运营与服务承载
叠加走：勾选多项时会提示——头部玩家（特斯拉）三条路同时走、互相加固，不是三选一。
① 制造驱动 · 比亚迪路线
定价权＝成本领先
  • 前提：有规模经济、成本差能持续拉开
  • 天花板低（组装层里称王）；风险：对手规模追上即被重挖
② 技术驱动 · 华为/小鹏路线
定价权＝差异化＋数据飞轮
  • 前提：智驾成决定性差异 + 数据飞轮真转得动
  • 复利最实；风险：同质化、数据边际递减
③ 服务驱动 · 出行服务商
定价权＝用户关系＋持续服务
  • 前提：L4 自动驾驶落地 + 用户接受"买服务不买车"
  • 最难最值钱（终局赢家）；风险：依赖 L4，最像愿景
勾选左侧筹码后给建议。
