GUI Agent
定义
GUI Agent 是负责图形界面操作的工具模块，能通过理解界面元素来完成操作任务（如在小程序中搜索、点击、填写等）。
核心问题
指令理解错误：把澄清指令当闲聊，不执行任务
工具参数混淆：创建任务 vs 补充信息参数混淆
任务难度高：GUI 操作涉及复杂界面交互，错误容忍度低
下游工具能力不足：如搜索能力差（搜"川菜馆"无结果，搜"川菜"才有）
