# 抖音处理快速指引（AI 侧）

本技能处理抖音视频时需要登录、持久化浏览器状态和基础依赖。仅在失败时参考本指南。

## 必备依赖
- Python 包：`patchright`
- 工具：`yt-dlp`、`ffmpeg`
- 模型：`whisper-cpp`（本地模型路径见 SKILL.md）

## 登录与浏览器状态
- 持久化目录：`data/browser_state/douyin_profile/`
- 登录方法：运行 `python3 scripts/douyin_login.py`，在弹出的浏览器中完成登录，按回车保存状态。
- 若提示“未检测到浏览器登录状态”或 cookies 过期，重新运行登录脚本。

## 链接与抓取要点
- 链接必须标准化为：`https://www.douyin.com/video/<数字ID>`（短链/modal_id 需先转换）。
- 抓取使用可见浏览器（非 headless），脚本自动监听 `aweme/v1/web/aweme/detail` 并保存为 `<输出根>/tmp/aweme_detail.json`。
- 若 60s 未抓到 detail，会自动刷新再等；仍失败多为登录/风控问题。

## 常用命令
- 下载音频：`python3 scripts/douyin_download.py <URL> --audio`
- 列出分辨率：`python3 scripts/douyin_download.py <URL> --list`
- 指定输出根：`--outdir <绝对路径>`（或在 `config.json` 配置 `output_dir`）

## 常见问题排查
1) **未捕获到 API 响应**  
   - 确认已登录（重新跑 `douyin_login.py`）。  
   - 确认链接为 `/video/<ID>`。  
   - 重试时保持浏览器可见，等待/刷新。
2) **yt-dlp 报需要 cookies**  
   - 说明登录状态未带上，重新登录；必要时用本机浏览器 cookies 运行。
3) **下载/转写失败**  
   - 检查模型/工具是否已安装；音频先转 mp3 再转写。  
4) **URL 过期**  
   - 抖音 CDN URL 有时效，获取后尽快下载。

## 安全提示
- `douyin_profile` 含登录凭证，不要提交/共享。需要重置时删除该目录后重新登录。
