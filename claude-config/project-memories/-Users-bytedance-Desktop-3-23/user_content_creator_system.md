王泽民 不只是豆包上车 AI PM，也在认真做 AI 内容创作者。关键事实：

- **X 账号**：2026 年 4 月末开始认真运营，到 2026-06 不到 1.5 个月涨到 ~1200 粉。内容 = 持续关注 AI 产品/Agent/开源工具/行业文章 → 提炼 → 短推/长推/文章输出。
- **核心诉求**：想要一套**低摩擦的内容生产系统**，支撑长期、稳定、高频输出（不靠一时兴奋、不每天从零硬扛）。这正是他做 article-forker / wiki 的深层动机之一——forks 不只是内化进 Planner 工作，也是喂养 X 输出的燃料。
- **成长地图（对他很有启发，可作沟通锚点）**：信息→知识→认知→**心→心性**（前三是他熟悉的，"心/心性"——发心/价值观/稳定性/真诚/长期认真——是他最受触动、想打开的更高维）。
- **认同的理念**：Ilya"对数尺度"（认知差距是维度差距，自己还在早期但空间大）；"意乐心"与"擅长>喜欢"（长期能做的事要有内在乐趣+内驱，不是硬"坚持"）。
- **认同的闭环**：AI 日报(高频输入)→周报(筛选聚合)→Wiki(分类沉淀)→原创认知/文章/手册；"以点带面再到球"的信息压缩与还原。2025 年 5 月加入某星球后一直践行输入-输出-成长闭环。

**Why**：他对 fork 产物反复不满的根因，是产物没服务到"内化进日常 + 喂养输出"这个闭环；他评判一切产出都看是否进入这个闭环。
**How to apply**：帮他做内容/知识整理时，默认目标是"既能内化进工作、又能成为可输出素材"，并保持低摩擦。关联 [[feedback_fork_workflow]] [[project_career_embodied_ai]] [[user_profile]]。
