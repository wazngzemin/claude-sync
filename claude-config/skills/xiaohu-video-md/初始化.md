# video skill 初始化指南

本文档仅在首次使用或依赖缺失时由 AI 读取，用于自动安装和配置所需依赖。

## 自动检测与安装

**首次使用时，AI 应自动检测并安装缺失的依赖，无需用户干预。**

### 依赖检测

```bash
# 必需工具
which yt-dlp ffmpeg

# 转写引擎（默认，二选一即可；模型会在首次转写时自动下载）
python3 -c "import mlx_whisper" 2>/dev/null && echo "mlx-whisper OK"        # Apple 芯片首选
python3 -c "import faster_whisper" 2>/dev/null && echo "faster-whisper OK"  # 通用首选

# 可选备份引擎 whisper-cli（不装也能用）
which whisper-cli 2>/dev/null
```

### 自动安装流程

**1. 安装必需工具**：
```bash
# 安装 Homebrew（如果未安装）
if ! command -v brew &> /dev/null; then
  /bin/bash -c "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/HEAD/install.sh)"
fi

brew install yt-dlp ffmpeg
```

**2. 安装转写引擎（默认，模型在首次转写时自动从 HuggingFace 下载，无需手动准备）**：
```bash
# Apple 芯片（M 系列）首选，走 Metal GPU 加速，最快
pip3 install --break-system-packages mlx-whisper

# 非 Apple 芯片 / 其它平台，用通用引擎
pip3 install --break-system-packages faster-whisper
```

**3.（可选）安装备份引擎 whisper-cli**：
默认引擎够用就**不用装这步**。仅当确实需要 whisper-cli（纯转文字最快、或上面两个都装不了）时才装，且要手动下 ggml 模型到 `~/.cache/whisper-cpp/`（与 SKILL.md 引用路径一致）：
```bash
brew install whisper-cpp

mkdir -p ~/.cache/whisper-cpp && cd ~/.cache/whisper-cpp
[ ! -f ggml-large-v3-turbo.bin ] && \
  curl -L -o ggml-large-v3-turbo.bin \
  "https://huggingface.co/ggerganov/whisper.cpp/resolve/main/ggml-large-v3-turbo.bin?download=true"
[ ! -f ggml-medium.bin ] && \
  curl -L -o ggml-medium.bin \
  "https://huggingface.co/ggerganov/whisper.cpp/resolve/main/ggml-medium.bin?download=true"
```

### 首次使用流程（完全自动化）

1. 用户请求视频处理
2. AI 检测依赖 → 发现缺失
3. AI **读取本文档并静默自动安装**（输出进度信息）
4. 安装完成 → 立即执行用户请求
5. 全程无需用户操作

### 依赖清单

- **必需工具**（Homebrew 安装）：
  - `yt-dlp` - 视频下载
  - `ffmpeg` - 音视频处理与烧字幕
- **转写引擎**（默认，二选一，pip 安装，模型自动下载）：
  - `mlx-whisper` - Apple 芯片首选，Metal GPU 加速
  - `faster-whisper` - 通用首选，CPU 即可
- **可选备份引擎**（不装也能用）：
  - `whisper-cpp`（命令 `whisper-cli`）+ ggml 模型（`ggml-large-v3-turbo.bin` 1.5GB / `ggml-medium.bin` 1.4GB，下到 `~/.cache/whisper-cpp/`）

### 注意事项

- 安装过程中输出简要进度信息，让用户了解状态
- 默认转写引擎的模型在首次转写时自动从 HuggingFace 下载，无需提前手动下
- macOS 以外系统：转写引擎用 `faster-whisper`（mlx 仅支持 Apple 芯片）；烧字幕字体需换成系统自带中文字体（Windows 微软雅黑 / Linux Noto Sans CJK）
- 受限资源时，先索要必要凭据（cookies/代理）再尝试
- YouTube 如遇 403/SABR/PO Token：优先用 `yt-dlp --cookies-from-browser chrome ...`（需要本机已安装对应浏览器且能读取其 cookies）

### 安装完成验证

```bash
which yt-dlp ffmpeg
python3 -c "import mlx_whisper" 2>/dev/null && echo "转写引擎 OK（mlx）" || \
  (python3 -c "import faster_whisper" 2>/dev/null && echo "转写引擎 OK（faster）")
```

全部检测通过后，返回主流程继续执行用户请求。
