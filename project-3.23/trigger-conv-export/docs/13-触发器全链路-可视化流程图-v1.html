⑮ 触发器频控
🎯 一句话把四段串起来(以"到公司提醒拿外卖"为例)
用户说"到公司提醒我拿外卖" → ②条件话 → 你在配置平台⑬注册成规则 → ①感知上行流持续把"距公司距离"送进Context → 你的⑮云端触发器每5秒判"距离<500m" → 命中 → 组装事件push出行Advisor → Advisor出建议 → Director出话术 → ④输出:TTS"快到公司了,别忘了拿外卖"。

你在这条链路里出现三次:注册(配置平台⑬) → 判定(云端触发器⑮) → 执行(端侧触发器㉔,如果规则下发端侧)。所有卡点都在"信号上来"和"你判定"的交界处。
