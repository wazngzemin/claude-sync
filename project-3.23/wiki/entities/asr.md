# ASR 语音识别

## 定义

Automatic Speech Recognition，将用户语音音频转换为文本的核心技术组件。在AI汽车中分为在线ASR、离线ASR和方言识别三种模式。

## 属性

| 属性 | 说明 |
|------|------|
| 在线ASR | 云端大模型识别，支持复杂语义理解，首字延迟≤200ms，尾字延迟≤500ms |
| 离线ASR | 本地模型识别，保障无网环境下的基础能力 |
| 方言识别 | 支持粤语、上海话、闽南语及强口音（陕西/山西/江淮/四川） |
| 二遍识别 | 第一遍实时上屏（低延迟），第二遍最终修正（高精度方言） |
| 热词增强 | 支持3000+特定词汇（通讯录、POI、媒资等）增强 |
| 地域POI增强 | 根据用户所在区域动态增加地点权重 |

## 性能指标

- 识别准确率（安静环境）：≥95%
- 识别准确率（嘈杂环境）：≥90%
- 首字延迟：≤200ms
- 尾字延迟：≤500ms
