## 摘要

Hassabis 认为当前范式"几乎确定"是 AGI 架构的一部分，但有 50% 概率还需要 1-2 个关键突破（持续学习、长程推理、记忆）。Agent 还不够好，根本原因是缺乏持续学习能力。DeepMind 正在将 AlphaGo/AlphaZero 时代的旧想法与现代基础模型结合。提出"爱因斯坦测试"作为真正智能的里程碑，以及"AlphaFold 式突破"的三个条件。对 AI 时代创业给出了防御性方向判断。

## 关键要点

- **AGI 进度判断**：当前范式是 AGI 的一部分，但 50% 概率还需 1-2 个关键突破
- **三个未解问题**：持续学习（continuous learning）、长程推理（long-range reasoning）、记忆（memory）
- **记忆的真实挑战**：百万 token 上下文只够 20 分钟实时视频；存储无限但检索成本不可忽视
- **Agent 的根本缺陷**：不能在具体使用环境中持续学习和适应 → 无法"交付后不管"
- **AlphaGo 思想回归**：DeepMind 正在重新审视蒙特卡洛树搜索等旧想法，在基础模型规模上重新应用
- **"爱因斯坦测试"**：用 1901 年物理知识训练系统，看能否产出 1905 年级别的发现（狭义相对论）→ 通过 = 具备发明新事物的能力
- **AlphaFold 式突破三条件**：①巨大组合搜索空间 ②清晰目标函数 ③足够数据或模拟器
- **小模型追赶速度**：前沿模型发布 0.5-1 年后，同等能力出现在端侧小模型
- **创业防御方向**：AI × 深科技交叉点（材料/医学/物理世界），不会被基础模型下次更新轻易取代

## 逐段拆解

### 第一段：AGI 进度审视

**功能**：校准认知 — 给出概率化判断

**深度分析**：Hassabis 的表述极其精确——"几乎确定是一部分"但"50%概率需要突破"。这比李广密的"2年内"保守得多。三个未解问题（持续学习/长程推理/记忆）不是"小bug"，而是架构级的缺陷。特别是记忆的分析：百万token=20分钟视频，这个具体化让"上下文窗口很大"的错觉瞬间破碎。

**隐含假设**：这些突破在可预见的未来是可能的（而非需要几十年）。

### 第二段：Agent 的根本问题

**功能**：诊断当前 Agent 为什么不够好

**深度分析**：核心诊断是"缺乏持续学习"——这意味着当前所有Agent产品本质上是"部署时能力冻结"的。这对你的 Planner 工作直接相关：Planner 也面临同样的问题——它不能从每次交互中真正学习和适应。"交付后不管"不可能是因为 Agent 不能自我进化。

**隐含假设**：持续学习问题一旦解决，Agent 能力会有质的飞跃。

**与你工作的直接关联**：Driver Agent 2.0 的 Planner 通过 memory 和 goal_list 做到了"有限记忆"，但本质上不是持续学习——SP 是固定的，模型不会因为使用而变得更好。

### 第三段：AlphaGo 思想回归 + 爱因斯坦测试

**功能**：指向下一个突破的方向

**深度分析**：两个关键洞察：（1）"旧想法+新规模"可能 = 突破。AlphaGo 的蒙特卡洛树搜索在 2016 年就存在，但当时没有大语言模型作为底座。现在把 MCTS 接到 LLM 上，可能释放新能力。（2）"Move 37 还不够，关键是能不能发明围棋"——区分了"在已知框架内创新"和"发明新框架"。爱因斯坦测试是对后者的形式化检验。

**隐含假设**：组合搜索 + 学习 + 规模 足以产生真正的创新能力。

### 第四段：AlphaFold 三条件

**功能**：给出可操作的判断框架 — 什么问题AI能突破

**深度分析**：三条件（大搜索空间 + 清晰目标函数 + 数据/模拟器）是一个极其实用的筛选器。可以用它来判断：你面前的任何问题，是否适合用"AI蛮力搜索"来解决。"物理定律允许它存在，剩下的就是如何高效找到"——这个世界观极其强大，把所有可解问题都变成了搜索问题。

**隐含假设**：所有有价值的科学问题都可以形式化为满足这三条件的搜索问题。

### 第五段：AI 时代创业方向

**功能**：行动指引

**深度分析**：两个关键判断：（1）小模型 0.5-1 年追赶大模型——意味着"用大模型能力做壁垒"的窗口极短；（2）最具防御性方向 = AI × 深科技交叉——因为"原子世界"的问题不会被模型更新轻易取代。结合 AGI 时间线：如果你的公司需要10年，AGI会在旅程中途出现——你必须把这算进商业计划。

**隐含假设**：纯软件/纯AI层的壁垒在AGI时代接近零。

## 论证链

"当前范式是AGI一部分但不完整"→"缺持续学习/长程推理/记忆"→"Agent因此无法交付后不管"→"AlphaGo旧想法+新规模可能是突破方向"→"爱因斯坦测试衡量真正智能"→"三条件判断什么问题AI能突破"→"创业应选AI×深科技交叉点"

## 隐含假设

1. 1-2个突破在"几年"内可实现（而非几十年）
2. 持续学习是Agent的根本瓶颈，解决后质的飞跃
3. 组合搜索+学习+规模 = 创新能力
4. 纯软件层壁垒在AGI时代趋近于零
5. DeepMind 的"旧想法+新规模"路线能跑通

## 观点边界

1. "50%概率需要突破"是Hassabis的个人判断，Anthropic/Meta可能有不同评估
2. AlphaFold 三条件对"无清晰目标函数"的问题（如社会系统、创意领域）不适用
3. "深科技交叉"方向需要10年，但如果AGI 2年就来（如李广密判断），这个建议就自相矛盾了
