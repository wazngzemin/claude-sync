（adapt）适应谐音
额-戴普特/əˈdæpt/
一（a）个人打（da）喷嚏扑通（pt）栽倒，新环境水土不服
需要慢慢适应
（adopt）采用谐音
额-道普特/əˈdɒpt/
打喷嚏之后采用药物治疗，药丸是圆（o）形的
用圆形 o 记住和 adapt 的差别字母
（bottleneck）瓶颈复合词
鲍头-耐克/ˈbɒtlnek/
瓶子（bottle）的脖子（neck）最细，水流到这里全堵住了
系统性能卡住的地方就是瓶颈
（sympathy）同情词根
辛帕思/ˈsɪmpəθi/
帅爷们（sym）感觉（path）到了，歪着头叹气（y）
他对别人的处境产生了同情
（empathy）共情词根
恩帕思/ˈempəθi/
走进去（em-）感觉（path）到了——不是旁观者的同情，是钻进对方身体里去感受
em- 是"进入"，进入对方的感觉 = 共情
（threshold）阈值谐音
斯瑞什-后德/ˈθreʃhoʊld/
三（thre）道报警线闪（sh）烁，hold住（hold）不许越过
只有突破这条线——阈值——才会触发报警
（latency）延迟谐音
雷腾-C/ˈleɪtənsi/
用户点了按钮，迟（late）迟等不到响应，盯着（n）转圈（cy）的 loading
接口响应太慢——这就是延迟
（deploy）部署谐音
迪-普洛伊/dɪˈplɔɪ/
深夜（de）值班，一行命令（ploy）敲下去，代码从本地飞到线上
服务上线——部署完成
（throughput）吞吐量复合词
斯鲁-普特/ˈθruːpʊt/
数据流穿过（through）管道被推出来（put），一秒过了多少条
管道每秒能推出多少——吞吐量
（rollback）回滚复合词
柔-拜克/ˈroʊlbæk/
上线出 bug，赶紧把代码卷（roll）回（back）去，版本倒退一格
紧急撤回——回滚
（iterate）迭代词根
伊特-瑞特/ˈɪtəreɪt/
一（i）条路（ter）走不通，再走（ate）一条——每次走出新路线
反复走新路线 = 迭代
（deprecate）弃用词根
戴普-瑞凯特/ˈdeprəkeɪt/
老接口额头上贴了一张（de）便签：「求（prec）你们别（ate）再调我了」
标记为不推荐使用——弃用
（resilient）有弹性的/高可用谐音
瑞-兹利恩特/rɪˈzɪliənt/
服务器被瑞（re）兹（zil）一拳打倒，弹（lient）了回来继续跑
打不死的服务——高可用
（idempotent）幂等的词根
爱登-剖腾特/aɪˈdempətənt/
同一个按钮（idem）按一次和按十次（potent），系统状态一模一样
重复操作结果不变——幂等
（paradigm）范式词根
派拉-代姆/ˈpærədaɪm/
一对（para）标杆模型（digm）竖在那里，所有后来者照着它的样子做
大家照着做的标准模型——范式
（quite）相当谐音
夸特/kwaɪt/
取（qu）出一根相当大的蜡烛（i 象形蜡烛），很特（te）别
相当特别
（quiet）安静的谐音
夸爱特/ˈkwaɪət/
和 quite 结尾的 et 和 te 顺序刚好相反
字母一调换，从"相当"变成"安静"
（mutex）互斥锁复合词
谬-泰克斯/ˈmjuːteks/
互相（mut-ual）排斥（ex-clusive），两个线程抢同一把锁，只有一个能进门
同一时刻只允许一个通过——互斥锁
（cache）缓存谐音
凯什/kæʃ/
收银台旁的快（ca）取架（che），最常买的东西放在伸手就够到的地方
把高频数据放在最近的地方——缓存
（scaffold）脚手架谐音
思凯-佛德/ˈskæfəld/
新项目刚起步，先搭（sca）一层（ff）骨架（old），代码模板自动生成目录结构
项目初始化的骨架——脚手架
