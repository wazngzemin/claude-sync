家里电脑 · 首次设置（只做一次，之后就一键了）
================================================

前提：家里这台 Mac 能登录你的 GitHub 账号（wazngzemin）。

【第一步】打开"终端"App
  启动台 → 搜"终端"（或 Terminal）→ 打开

【第二步】把下面这一行粘进终端，回车（把仓库拉到本机）：

    git clone git@github.com:wazngzemin/claude-sync.git ~/.claude-sync

  如果报 SSH/Permission 相关错误，改用这一行（会让你在浏览器登录 GitHub）：

    git clone https://github.com/wazngzemin/claude-sync.git ~/.claude-sync

【第三步】克隆完成后，再粘这一行回车，一键安装：

    bash ~/.claude-sync/家里-一键拉取.command

  装完，打开 Claude Code 就是办公室那台的样子了
  （配置、25个技能、记忆、3.23全部工作文件，都到位）。

================================================
以后每次同步（办公室点了推送之后）：

  打开访达 → 前往文件夹（⌘⇧G）→ 输入 ~/.claude-sync
  → 双击里面的「家里-一键拉取.command」

  它会自动 git pull 拉最新 + 装到本机。就这么简单。
