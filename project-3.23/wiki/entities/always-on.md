# Always On 全时监听模块

## 定义

端侧持续运行的监听模块，负责全时多模态聆听（音频流+视频流），进行情景理解、事件发现和弱网/无网兜底，是AI汽车从"交互驱动"向"感知驱动"演进的核心基础设施。

## 属性

| 属性 | 说明 |
|------|------|
| 音频流监听 | 全时多音区音频采集与处理 |
| 视频流监听 | 舱内摄像头持续感知 |
| 情景理解 | 舱内人物/行为/环境描述 |
| 事件发现 | 注册事件监测、行为事件捕获 |
| 弱网兜底 | 端侧算力独立运行，不依赖网络 |
| 挂起任务监听 | 持续监听车辆状态以触发挂起任务 |

## 关键输出

- 工程化端状态信号
- 多模态短期记忆
- 触发[[advisor]]主动服务的事件流
