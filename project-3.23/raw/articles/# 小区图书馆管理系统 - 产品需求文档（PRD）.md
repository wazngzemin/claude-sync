## 一、项目概述

本项目为**小区图书馆**打造一款简洁高效的图书管理系统，实现图书在线查看、借阅自动管理、逾期智能提醒的数字化图书馆运营。

### 核心价值
- 让居民方便地了解图书馆有哪些书籍
- 管理员高效管理借阅流程
- 自动化计算逾期费用，避免人工核算错误

| **核心痛点** | 1. 纸质记录借阅信息繁琐、易丢失<br>2. 无法快速查看有哪些书<br>3. 人工追踪还书日期容易遗漏<br>4. 逾期扣费计算麻烦 |

## 三、产品路线图

### V1: 最小可行产品 (MVP)

#### 3.1 图书管理
- [x] 添加图书（书名、作者、楼层、书籍类型）
- [x] 查看图书列表，支持搜索（按书名/作者/类型）
- [x] 编辑/删除图书

#### 3.2 借书人管理
- [x] 注册借书人（身份证号码、姓名、手机号、押金）
- [x] 查看借书人列表
- [x] 编辑/删除借书人

#### 3.3 借阅管理
- [x] 借书（选择图书 → 选择借书人 → 管理员手动设定还书日期）
- [x] 还书（自动计算逾期天数和扣费金额）
- [x] 扣费记录（记录每次逾期扣费明细）
- [x] 当前借阅列表（查看正在借出的图书）

#### 3.4 提醒与首页
- [x] 首页仪表盘（显示待还书数量、逾期数量）
- [x] 逾期高亮（逾期未还的图书红色标记）
- [x] 借阅历史（查看所有借阅记录）

#### 3.5 数据存储
- [x] 本地存储（浏览器 LocalStorage）

---

### V2 及以后版本

- [ ] 数据导出（导出图书列表、借阅记录为 Excel/CSV）
- [ ] 批量操作（批量添加图书、批量导入借书人）
- [ ] 统计报表（借阅次数统计、最受欢迎图书排行）
